Purpose of the Project:
-----------------------
-This project is a Demo for automation framework developped in java where Maven is used 
as the build automation tool, alongside MavenSureFire Plugin for test cases management.
-Page Object Model concept is implemented, Selenium and TestNG 
frameworks are used for implementing some web testing test cases.
-Log4j is used for logging and TestNG Listeners concept implemented for Screenshots taking on test
failure.
-ExtentReporter is used for reporting.
-Project originally developed on Eclipse.
